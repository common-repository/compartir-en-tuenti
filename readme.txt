Permite colocar un boton para compartir en Tuenti junto al contenido de tu blog, permitiendo personalizarlo

== Description ==

* Se trata de un plugin sencillo, que se limita a colocar un boton para compartir en Tuenti junto al contenido de tu blog que tu selecciones. Cuenta con bastante personalizacion.
* Si encontrais algun error o teneis sugerencias no dudeis en dejar un comentario en http://www.tecnocaos.com
* Si te gusta, por favor, vota positivamente!


== Options ==
* Permite seleccionar el tipo de boton que queremos que se muestre.
* Podemos seleccionar la posicion del boton respecto al texto
* Ahora se puede modificar el CSS del boton
* Es posible elegir los lugares donde va a aparecer, entre portada, entradas y paginas
* Posibilidad de añadir nofollow a los enlaces
* Es posible actualizar tu estado de Tuenti automaticamente con enlaces a tus nuevas entradas del blog


== Frequently Asked Questions ==
= ¿Como puedo cambiar la configuracion del boton? =
Accediendo al menu de ajustes y clickando en el enlace a Compartir en Tuenti

== Installation ==
* Coloca el directorio /compartir-en-tuenti en /wp-content/plugins/ de tu blog.
* Activalo desde el menu de plugins de tu panel de administrador de Wordpress.
* Personalizalo a tu gusto en el menu de ajustes.
